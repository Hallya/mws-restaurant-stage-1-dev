# PWA Restaurant Reviews (Client Side)
---

#### Installation

Run in your terminal:
1. `git clone https://github.com/hallya/restaurant-reviews.git`
2. `cd restaurant-reviews`
3. `npm i`
4. `npm start`
You are all set !
